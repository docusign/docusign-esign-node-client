(static) getEnvelopes(envelopeType, doFullRetrieval, callback)

Get information about envelopes for the account with the given `apiToken`.
Parameters:
Name Type Description
envelopeType string The type of envelope to get. Valid values are 'all', 'awaiting_my_signature', 'out_for_signature', 'completed', and 'drafts'.
doFullRetrieval boolean If true, retrieve all envelopes of the given `envelopeType`. Otherwise, only get the first 50 most recent envelopes.
callback function Returns in the form of function(error, response) such that envelopes will live at `response.folderItems.
